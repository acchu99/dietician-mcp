Deployment Guide

Local Development

1. Install Dependencies

pip install -r requirements.txt

2. Configure Environment

Set required environment variables:

export PORT=8000
export MONGODB_URI=your_connection_string

3. Start Server

python main.py

Docker Deployment

Standard Deployment

docker compose up --build -d

Builds and starts the service in detached mode

Test-Driven Deployment

./scripts/test-and-run.sh

Runs automated tests before deployment. Deployment only proceeds if all tests pass.

Manual Test Execution

docker compose -f docker-compose.test.yml up --build --abort-on-container-exit

Runs tests in isolated containers with automatic cleanup

Testing

Run the test suite locally:

pytest -v

Test coverage includes:

  • Server initialization and health checks
  • Tool registration and execution
  • API endpoint responses
  • Error handling and edge cases
